## Tech Stack

**Client:** Expo, React-Native, React, Firebase

**Database:** Firestore

## Run Locally

Clone the project

```bash
  git clone https://github.com/abhirupkumar/Instagram-clone.git
```

Go to the project directory

```bash
  cd Instagram-clone
```

Install dependencies

```bash
  npm install
```
